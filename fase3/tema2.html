Listas

Se pueden representar elementos en forma de lista dentro de nuestros documentos de forma sencilla y con versatilidad. Estas listas podran incluir cualquiera de los elementos html e incluso podemos definir listas anidadas, es decir listas dentro de listas. HTML nos permite crear tres tipos de listas.

  • Listas no numeradas
  • Listas numeradas
  • Listas de definicion
  1. Listas no numeradas

    2. Con este tipo de lista podemos especificar una serie de elementos sin un tipo de orden predeterminado, precedidos por una marca o viñeta que nostros mismos podemos definir.

  2. Listas numeradas

    3. Con este tipo de lista se puede especificar una serie de elementos numerados según el lugar que ocupaan en la lista.

  3. Listas de definicion

    4. Estas listas nos van a servir para especificar una serie de terminos y sus definiciones correspondientes.
