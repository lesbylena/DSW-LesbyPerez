Proposito del sitio web

Este sitio web fue creado como herramienta didactica con el proposito de elaborar recursos educativos digitales como las tecnologias HTML5, CSS3, que le permitan a las instituciones educativas dinamizar las estrategias de aprendizaje, enriqueciendo el conocimiento humano y supliendo una necesidad de presencialidad a causa de muchos inconvenientes, dentro de ellos la pandemia del covid-19.
