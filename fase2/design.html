Flexbox

Es un módulo de diseño de caja flexible, más comúnmente conocido como flexbox, herramienta poderosa para crear diseños con CSS. Flexbox es un diseño unidimensional que puede colocar elementos horizontal o verticalmente (pero no ambos).

Referencias: Attardi, J. (2020). Flexbox. En Modern CSS (págs. 205-228). Apress, Berkeley, CA. https://link.springer.com/chapter/10.1007/978-1-4842-6294-8_10

Listas

Se pueden representar elementos en forma de lista dentro de nuestros documentos de forma sencilla y con versatilidad. Estas listas podran incluir cualquiera de los elementos html e incluso podemos definir listas anidadas, es decir listas dentro de listas. HTML nos permite crear tres tipos de listas.

Referencias: Muñoz, V. J. E. (2012). HTML, presente y futuro de la web. Vicente Javier Eslava Muñoz. Enlaces
